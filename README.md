# 🚀 JavaScript Assignment Practice 08

A JavaScript assignment repository focused on **Asynchronous JavaScript** through practical coding exercises and real-world programming scenarios.

This repository contains **12 assignment questions** and **1 bonus interview question** covering callbacks, timers, promises, async/await, the Fetch API, and the JavaScript Event Loop.

---

## 📚 Topics Covered

- Synchronous vs Asynchronous
- Callbacks
- setTimeout()
- clearTimeout()
- setInterval()
- Promise
- Promise Chaining
- Async / Await
- Event Loop
- Microtask Queue
- Callback Queue
- Fetch API
- JSON Parsing
- Error Handling
